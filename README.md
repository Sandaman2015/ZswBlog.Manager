# ZswBlog Manager

### 项目概述：

此项目为ZswBlog的后台管理系统，拥有对博客的基本配置和查看，满足一定的个性化需求

该项目通过vue-admin-template生成，内置了markdown编辑器发布文章

支持图片上传到OSS（需要后端服务支持）

[ZswBlog前端](https://github.com/Sandaman2015/ZswBlog3.0)

[ZswBlog后端服务](https://github.com/Sandaman2015/ZswBlog.Core)
